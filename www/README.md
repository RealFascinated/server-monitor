# Monitor Web

Dashboard for Monitor. TanStack Start, React, TypeScript, shadcn/ui.

**Users:** install the full stack with [Docker](../README.md#install-with-docker).

**Developers:**

```bash
bun install
cp .env.example .env
bun dev
```

Runs on port 3000.

```bash
docker build -t monitor-www .
```

Image: `ghcr.io/realfascinated/monitor-www`
